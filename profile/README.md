# 🧑🏼‍💻 How To Start
🔗 [www.iailog.store](https://www.iailog.store)

- **Android** : APK 파일을 직접 다운로드 받아 실행할 수 있습니다.  
- **iOS** : 실제 앱 설치는 불가능하며, 대신 **시연 영상**을 통해 확인할 수 있습니다.
