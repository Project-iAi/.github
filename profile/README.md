## 🌟 소개
맞벌이 가정에서 부모가 **아이의 감정**을 놓치지 않도록,
- 아이는 **AI 캐릭터**와 **음성 대화**로 하루를 이야기하고  
- 서비스는 **STT/TTS + GPT 요약**으로 자동 기록/요약,  
- 부모는 **캘린더/감정 흐름/그림일기**로 **한눈에 확인**합니다.

> 기존 교사 기록 중심의 앱과 달리 **아이의 주관적 감정**과 **목소리/언어 표현** 자체가 기록의 중심입니다.

## ✨ 핵심 기능
- **온보딩 & 캐릭터 선택**: 아이 프로필, 보호자 정보 입력 → 캐릭터/대화 스타일 선택
- **음성 대화 기록**: STT로 텍스트 변환, GPT 요약으로 핵심 내용 정리
- **오늘의 감정**: 간단한 감정 선택(😀 😐 😢 등) + 이유 말하기
- **캘린더**: 일/주/월별 기록과 감정 변화 시각화
- **그림일기 자동 생성**: 대화 요약본 → 이미지 생성
- **요약 리포트**: 부모에게 주간/월간 요약 
- **(선택) 소셜 로그인**: 카카오 등 OAuth 연동
